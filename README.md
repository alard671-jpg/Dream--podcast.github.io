# Dream Podcast - Website (GitHub Pages ready)

محتويات المشروع:
- صفحات: index.html, about.html, programs.html, news.html, contact.html
- مجلدات: css/, js/, images/

**كيفية النشر على GitHub Pages**
1. أنشئ Repository جديد على GitHub.
2. ارفع محتويات هذا المجلد (أو اسحب commit).
3. فعّل GitHub Pages من الإعدادات، اختر الفرع `main` و/أو `gh-pages` ثم `/root`.
4. سيظهر الموقع على `https://USERNAME.github.io/REPO_NAME`.

**تعديلات لاحقة**
- يمكنك تعديل أي ملف HTML/CSS/JS مباشرة عبر GitHub أو تحرير الملفات محليًا وإعادة رفعها.
- النموذج في صفحة "تواصل" يفتح بريدك الإلكتروني (mailto). لرسائل عملية تلقائية يلزم خادم بريد أو خدمة مثل Formspree.

**ملاحظات تقنية**
- البث المضمن (Zeno.FM) مجاني طالما Zeno يسمح بالتشغيل عبر iframe؛ هذا ليس موقعنا سيطلب اشتراك.
- تم استبدال شعارات المتاجر بزر "استمع الآن" أنيق حسب طلبك.
- الروابط الاجتماعية والبريد/الهاتف مُعدّة كما طلبت.

إذا تريد: أجهز لك مستودع GitHub جاهز (أقوم بتهيئة ملفات README و .gitignore) — أرسل اسم المستخدم واسم المستودع إن رغبت.
